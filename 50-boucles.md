Une **boucle de requête** (Query) peut être créée dans l'interface de Bricks sur n'importe quel élément bloc.

## Usages avancés

### Filtres de requêtes

Il est possible de [filtrer des requêtes](https://academy.bricksbuilder.io/article/query-filters/).

C'est ainsi que le site [fontes.eracom.ch](https://fontes.eracom.ch/) a été construit.

### Enregistrer des requêtes

Il est possible d'enregistrer une Query pour la rendre réutilisable - voir [Global Queries & Query Manager](https://academy.bricksbuilder.io/article/global-queries-query-manager/).
